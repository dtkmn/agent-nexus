# Screenshots

## Main UI - Agent Graph Visualization

![A2A Gateway UI](../assets/ui-screenshot.png)

The main interface shows:
- Interactive force-directed graph of all agents
- Real-time updates (manual or auto-refresh every 5s)
- Color-coded agent types
- Draggable nodes for custom layout

## Agent Details Panel

When clicking on an agent node, a details panel appears showing:
- Agent ID and name
- System prompt preview
- Enabled tools
- Downstream peer agents

## Multi-Agent Demo

The demo setup creates a support orchestration system:
- **Support Orchestrator**: Routes customer inquiries
- **Billing Agent**: Handles pricing and payment questions
- **Technical Agent**: Answers integration and API questions
- **Sales Agent**: Discusses features and enterprise plans

## Graph Features

- **Legend**: Shows agent type color coding
- **Status Indicator**: Shows system health
- **Peer Badges**: Orchestrators show count of downstream agents
- **Hover Effects**: Enhanced shadows on mouse over
- **Click to Details**: Click any node to see full agent configuration
